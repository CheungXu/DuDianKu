# 2026-03-31 DuDianKu 开发日志

## 进展

### 牍层完整实现

已完成牍层基础功能和写入接口：

**文件结构**：
```
module/du/
├── __init__.py      # 模块入口
├── models.py        # DuRecord (28字段)
├── database.py      # SQLite数据库管理
├── manager.py       # CRUD + 写入接口 + 回调
├── test_du.py       # 测试脚本 (全部通过)
└── README.md        # 设计文档
```

**写入接口**：
- `insert_send(content, agent_id)` - Agent发送消息入库
- `insert_receive(content, agent_id, sender_id)` - Agent接收消息入库
- `insert_batch(records)` - 批量导入历史数据

**回调机制**：
- 构造函数回调: `on_insert`, `on_batch`, `on_error`
- Hook装饰器: `@manager.hook('after_insert')`

**热度管理**：
- access_count 访问计数
- heat_score 热度评分 (0-100)
- heat_level 热度等级 (hot/warm/cold)
- decay_factor 衰减因子 (30天衰减周期)
