# 牍典库 | DuDianKu
**积牍为文，汇典成库，铸Agent恒久之思**

专为 AI Agent 打造的**层级化长期记忆 + 结构化知识库**开源框架

---

## 项目简介
牍典库（DuDianKu）是一款极简、可扩展的 Agent 记忆与知识管理框架，以中式古典知识体系为设计理念，构建**「牍 → 典 → 库」**三层递进架构，完美匹配原始数据→知识凝练→体系化存储的全流程，解决 AI Agent 短期记忆遗忘、知识碎片化、无法长效沉淀的核心问题。

框架轻量易用，可无缝集成到各类智能体、大模型应用中，成为 Agent 的**长效知识大脑**。

## 核心理念
恪守「从单元到体系」的知识生长逻辑，三层架构一一对应知识进化全链路：
1. **牍 (Du) - 最小记忆单元**
   原始数据载体，对应 Agent 的单条交互记录、碎片化信息、感知输入、原始文本等最基础的记忆颗粒。
2. **典 (Dian) - 凝练知识范式**
   对「牍」进行自动总结、萃取、去重、归纳，形成标准化的经验、规则、知识点与核心结论，完成从数据到知识的升华。
3. **库 (Ku) - 体系化知识底座**
   将所有「典」结构化存储、索引、检索与复用，构建 Agent 专属的长期知识库，实现知识的持久化与全局共享。

## 架构设计
```
原始数据/交互记忆
        ↓
┌─────────────────┐
│    牍 (Du)      │  最小单元：原始记忆、碎片化数据
└─────────────────┘
        ↓ 自动萃取、编纂
┌─────────────────┐
│    典 (Dian)    │  凝练知识：经验、规则、核心结论
└─────────────────┘
        ↓ 归集、结构化
┌─────────────────┐
│    库 (Ku)      │  知识底座：持久存储、检索复用
└─────────────────┘
```
