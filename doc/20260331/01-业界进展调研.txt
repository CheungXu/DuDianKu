                    大模型长期记忆系统 - 业界进展调研报告
================================================================================

调研日期: 2026-03-31
调研目的: 确定 Agent 长期记忆系统的业界进展和技术方案

================================================================================
一、向量记忆派（主流方案）
================================================================================

1. Mem0 (mem0.ai)
   - 最流行的 Agent 记忆方案
   - 向量存储 + 自动提取关键信息
   - 支持用户/会话/Agent 级记忆隔离
   - 已集成到 LangChain, AutoGen, CrewAI 等主流框架

2. Letta (原 MemGPT)
   - UC Berkeley 团队出品
   - 核心创新: 虚拟上下文管理，模拟"有限记忆窗口"
   - 分层记忆: 核心记忆(Core)、工作记忆(Working)、档案记忆(Archive)
   - 让 Agent 自己管理记忆的"换入换出"

================================================================================
二、结构化分层派（与 DuDianKu 理念最契合）
================================================================================

1. Smart Agent Memory ★推荐
   - 三层存储架构:
     * Markdown - 人可读，QMD 可搜索
     * JSON - 结构化元数据
     * SQLite/FTS5 - 高性能全文搜索
   - 温度模型 + Skill 经验记忆 + 自动归档
   - 纯 Node.js，零外部依赖

2. Sulcus ★推荐
   - 热力学记忆模型: 温度衰减 + 间隔重复
   - 记忆有"热度"，重要记忆保持热，冷记忆逐渐淡出
   - 反应式记忆召回，适合长期对话场景

3. Neural Memory
   - 联想记忆 + 激活扩散模型
   - 记忆之间有连接，激活一个会带动相关记忆
   - 类似人脑的"回忆联想"机制

================================================================================
三、知识图谱派（上层知识体系）
================================================================================

1. Graphthulhu
   - 知识图谱 MCP server，支持 Logseq/Obsidian
   - 37 个工具: 读/写/搜索/分析"第二大脑"
   - 记忆节点 + 关系边，构建知识网络

2. ScholarGraph
   - 学术文献知识图谱
   - 多源论文搜索 + AI 辅助分析 + 知识图谱构建

3. Memory Graph
   - 可视化记忆图谱，Luhmann 编码
   - 交互式探索 + 叙事叠加 + 导出功能

================================================================================
四、一体化方案
================================================================================

1. Memex ★推荐
   - 统一记忆插件: 对话记忆 + 文档搜索
   - 90% E2E 准确率（LongMemEval，ICLR 2025 论文）
   - 单 SQLite 存储，轻量高效

2. OpenClaw Memory OS
   - 数字永生服务愿景
   - 对话记录基础设施（Phase 1）
   - 知识管理 + 认知连续性

3. Agent Knowledge Capture
   - 统一知识捕获: URL/视频/论文/社交帖子/Agent 输出
   - "任何值得找回的东西都能保存"

五、业界趋势总结
================================================================================

┌──────────┬────────────────────┬─────────────────────────────────┐
│   层级   │   DuDianKu 理念    │           业界对应方案           │
├──────────┼────────────────────┼─────────────────────────────────┤
│   底层   │   原始数据记录     │ Memex/Smart Agent Memory 原始层  │

主流演进方向:

    1. 分层存储
       不是单一向量库，而是"热/温/冷"分层

    2. 自动萃取
       从原始对话自动生成结构化记忆

    3. 知识图谱化
       记忆之间建立关联，形成知识网络

    4. 温度/衰减模型
       模拟人类记忆的"遗忘曲线"
